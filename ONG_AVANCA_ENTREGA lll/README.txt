ONG AVANCA - Entrega III (Arquivos separados)
Arquivos:
- index.html (Início)
- projetos.html (Projetos; grid renderizado via js/script.js)
- cadastro.html (Formulário validado, armazenado em localStorage)
- css/style.css (tema roxo/magenta/azul elétrico)
- js/script.js (tema, validação e render de projetos)
- img/ (placeholders)

Como usar:
- Extraia o ZIP e abra index.html no navegador (ou use Live Server no VS Code).
- A navegação é tradicional entre arquivos (links HTML).
- Dados do formulário são salvos em localStorage (chave 'registrations').
